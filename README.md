# FXGL-Gradle
A minimal FXGL project using gradle

## Build

On Linux / Mac OS X

```
./gradlew build
```

On Windows

```
gradlew build
```

## Run

On Linux / Mac OS X

```
./gradlew run
```

On Windows

```
gradlew run
```

## Standalone game executable (`.jar`)

After building the game you will find the executable in `build/libs/` with a suffix `-all`.
